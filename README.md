# overview
設定ファイル集

# 使い方
- .gitconfig_alias
  - .gitconifに以下を追加してinclude
  - ```
    [include]
      path = ~/hako_dotfile/.gitconfig_alias
    ```

- .tmux.conf
  - .tmux.confに以下を追加してinclude
  - ```
    source-file ~/hako_dotfile/.tmux.conf
    ```

- alias.fish 
   - fishのalias設定ファイルに以下を追加してinclude
   - ```
      source ~/hako_dotfile/alias.fish
      ```
